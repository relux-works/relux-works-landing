### Rules for sub-agents:
- Import and use `t()` helper from `src/i18n/utils.ts`
- Use CSS variables from `palette.css` (no hardcoded colors)
- Use `.container`, `.section`, `.section-title` from global.css
- Component-scoped styles via `<style>` tag in Astro
- Mobile-first responsive
- Receive `lang` prop, pass to `t(lang, 'key')`
